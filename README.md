# tic-tac-toe
Tic Tac Toe game. Combination of HTML, JavaScript, and CSS.

# How to

1) Load files main.html, game.css, my_script.js into folder. <br>
2) Open main.html file in a browser. <br>
3) Enjoy the game! <br>
